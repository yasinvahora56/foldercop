# FolderPop

FolderPop is a fast Windows search and quick-launch utility designed for instant file and folder access.
Built with Rust and Tauri for performance, reliability, and low resource usage.

## Highlights

* ⚡ Fast local search with indexed results
* 🗂️ Instant file and folder discovery
* 🎯 Filters for apps, images, videos, audio, PDFs, and documents
* ⭐ Favorites and quick folder access
* ⌨️ Global shortcut support (`Win + F2`)
* 🔄 Secure automatic updates
* 🪶 Lightweight and optimized for Windows

## Download

Download the latest version from Releases:

Available builds:

* Installer (.exe)
* Portable (.zip)

## Installation

### Installer

1. Download the latest `.exe`
2. Run the installer
3. Launch FolderPop

### Portable

1. Download `.zip`
2. Extract anywhere
3. Run `FolderPop.exe`

No installation required.

## Auto Updates

FolderPop checks for updates automatically and installs signed releases securely.
